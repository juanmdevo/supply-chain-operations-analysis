# supply-chain-operations-analysis

## Project Overview
This project analyzes supply chain and logistics performance to identify delivery delays, regional bottlenecks, and opportunities for operational optimization.

The analysis combines Python for data cleaning and exploratory analysis with Power BI for interactive dashboarding, delivering business-ready insights and strategic recommendations.

## Objectives 
- Evaluate shipping performance across different delivery modes
- Identify regions with high delivery delays (bottlenecks)
- Analyze profitability by shipping method and region
- Provide data-driven recommendations to improve efficiency and reduce delays

## Tools & Technologies
- Python (pandas, matplotlib)
- Power BI
- DAX (Data Analysis Expressions)
- Excel / CSV data

## Key Insights

- **Standard Class** shipping delivers the highest profit with the lowest delay
- **Second Class** shows the worst performance with the highest delivery delays
- **First Class** is profitable but operationally inefficient
- **Central Asia** and **Central Africa** are major bottleneck regions
- High sales regions include **Western Europe** and **Central America**

## Business Recommendations

- Optimize shipping strategy by reducing reliance on **Second Class**
- Shift volume toward **Standard Class** for better reliability and profitability
- Improve **First Class** delivery performance through operational audits
- Prioritize logistics improvements in high-delay regions
- Expand focus on high-performing sales regions
